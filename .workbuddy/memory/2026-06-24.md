# 2026-06-24

## 修复斯巴鲁任务鼠标点击不稳定

**问题**: `_win32_click_at` 中鼠标按下和抬起之间只有 10ms 间隔，游戏经常来不及注册点击，导致 click_match 步骤有时成功有时失败。

**修改** (`core/task_base.py`):

1. `_win32_click_at` 签名从 `(cx, cy, hover_delay)` 改为 `(cx, cy, button, click_delay)`:
   - `button`: 支持 left/right/middle，不再硬编码左键
   - `click_delay`: 鼠标按下到抬起的间隔，由 `step.delay` 传入（如 500ms），形成完整按下-保持-抬起周期
   - 悬停时间从 `hover_delay`(step.delay, 500ms) 缩短为固定 100ms（只需让游戏注册光标位置）

2. 三处调用点均更新为新签名，传入 `button=step.button` 和 `click_delay=step.delay`
